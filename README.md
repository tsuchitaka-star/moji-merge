# モジマージ

落ちてくる言葉ブロックを合体させて長い言葉に育て、タイピングで消してスコアを稼ぐ個人向けカジュアルゲームです。

## コンセプト

- スイカゲーム風の落ちもの合体パズル
- 言葉ブロックを育てるほど高得点
- 大きいブロックほど盤面を圧迫し、入力する言葉も長くなる
- 小さいうちに安全に消すか、育てて高得点を狙うかを選ぶ
